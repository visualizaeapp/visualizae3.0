# **App Name**: Visionary Canvas

## Core Features:

- Canvas Interaction: Provides a primary Nextjs client page for canvas display, a navigation header and simple editing toolbar.
- Layer Management: Lets user manage visual layers such as image transforms and compositions.
- Image Generation: AI tool that generates or modifies images based on text prompts and reference images, enhancing the creative process with AI-driven content.
- Image Enhancement: Enhance a group of target images based on prompt and styles from reference images, delivering refined results via server actions.
- Firebase User Auth: Firebase authentication with login and authentication tokens with persist using Google OAuth.
- New Visual Project: Generate and manage visual images and compositing via Nextjs server and client actions
- Action Management: Support use viewport size tools including maximize, restore or tile based selection tools such as rectangle selection to apply different sizes to individual layers to control overall composite image quality

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5), providing a sense of calm, intelligence, and sophistication.
- Background color: Very light grayish-blue (#F9FAFB) creates a clean, spacious backdrop.
- Accent color: A soft lavender (#D8B4FE) used sparingly to highlight interactive elements and important information.
- Font pairing: 'Inter' for both headings and body text; a clean sans-serif font known for its readability and modern aesthetic.
- Simple, line-based icons to maintain a modern, uncluttered feel. Use icons with rounded corners to soften the design.
- Emphasize whitespace to create a sense of openness and clarity. Use a grid system to align elements and maintain a balanced look.
- Subtle, smooth transitions for user interactions. Employ fade and slide animations to make interactions feel fluid.
